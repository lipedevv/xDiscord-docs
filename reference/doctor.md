# Doctor

Use:

```text
/xdiscord doctor
```

Permission:

```text
xdiscord.admin
```

The doctor command checks:

* xDiscord version.
* xCore status.
* Database connection and type.
* Discord bot enabled state.
* Token presence.
* Bot connection.
* Guild availability.
* Verified role availability and role hierarchy.
* PlaceholderAPI.
* LuckPerms when VIP sync is enabled.
* Discord whitelist state.
* World restrictions state.
* Total linked accounts.

It is meant for quick setup debugging after editing config files or restarting the bot.
